# Personal Library Manager📚

A simple Python project to manage a personal book collection and wishlist.

## Features
- Add books to your library
- Add books to a wishlist
- Move books from wishlist to library when acquired
- Remove donated books from your library
- Display updated lists after each action

## How it works
The program takes user input to manage two lists:
- Books you own
- Books you wish to have

It allows updating both lists dynamically based on user actions.

## Language
Python
